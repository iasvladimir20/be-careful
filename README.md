# Be Careful app
El presente repositorio contiene el desarrollo de la aplicación móvil de Be Careful para la plataforma de iOS.
## Requisitos
* Xcode 11.4 en adelante.
* iOS 13 en adelante.
* Cocoapods
* Swift 5
